## ğŸš€ Getting Started

### **Prerequisites**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS/JS (for customization)
- Text editor (VS Code, Sublime Text, Atom, etc.)

### **Installation**

1. **Clone the repository:**
```bash
git clone https://github.com/Manuella-R/portfolio.git
cd portfolio
```

2. **Open in browser:**
```bash
# Simply open index.html in your browser
open index.html  # Mac
start index.html  # Windows
xdg-open index.html  # Linux
```

3. **Or use a local server (recommended):**
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

4. **Access the portfolio:**
```
Open http://localhost:8000 in your browser
```

### **Quick Start**
No build process required! The portfolio runs with pure HTML, CSS, and JavaScript.

## ğŸ¨ Customization

### **Personal Information**

#### 1. Update Profile Details (index.html)
```html
<!-- Home Section -->
<h3 class="hello">Hello! My name is <span class="name">Your Name</span></h3>
<p>Your professional summary here...</p>

<!-- About Section -->
<div class="info-item padd-15">
    <p>Email : <span>your.email@example.com</span></p>
</div>
```

#### 2. Replace Profile Image
- Add your photo to `/images/` folder
- Name it `YourName.jpg` or update reference in HTML:
```html
<img src="images/YourName.jpg" alt="Your Name">
```

#### 3. Update CV
- Replace `/cv/Rehema.pdf` with your CV
- Keep the same filename or update the link:
```html
<a href="cv/YourCV.pdf" download>Download CV</a>
```

## ğŸ“§ Contact Form Setup (Optional)

The portfolio includes a contact form HTML ready for backend integration.

### **Backend Options:**

#### 1. **Formspree** (Easiest)
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
    <!-- form fields -->
</form>
```

#### 2. **EmailJS** (No Backend)
```javascript
emailjs.send("service_id", "template_id", {
    from_name: name,
    message: message,
    reply_to: email
});
```

#### 3. **Node.js Backend** (app.js included)
```bash
npm install express nodemailer body-parser
node app.js
```

Update `app.js` with your email credentials:
```javascript
auth: {
    user: 'your-email@gmail.com',
    pass: 'your-app-password'
}
```

#### 4. **PHP Backend**
```php
<?php
$to = "your-email@example.com";
$subject = $_POST['subject'];
$message = $_POST['message'];
mail($to, $subject, $message);
?>
```
